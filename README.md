# Trouve ton artisan 
 
## 📦 Prérequis 
- Node.js 
- MySQL 
- VS Code 
 
## 🚀 Installation 
1. Cloner le repo 
2. Installer les dépendances dans `/client` et `/server` 
3. Lancer le front-end : `npm start` 
4. Lancer le back-end : `node index.js`
